# AI记账 💰

纯离线的智能记账 App，输入自然语言自动识别金额、人员、分摊方式。

> 支持 Android APK 安装，无需网络，数据存于手机本地。

## 功能

- 🧠 **AI 解析** — 输入 "晚饭三人AA 120元" 自动识别金额、品类、人数、分摊
- 👥 **多人分摊** — 支持 AA 均摊 / 单人 / 按比例
- 💱 **多币种** — 美元、欧元、日元……自动换算人民币
- 📅 **日历视图** — 按天查看每日支出
- 📊 **月度统计** — 按分类汇总个人支出
- ✏️ **编辑/删除** — 长按账单可编辑或删除
- 📋 **剪贴板识别** — 打开 App 自动检测支付相关文字

## 安装

下载 [最新 APK](https://expo.dev/accounts/jin121/projects/ai-bookkeeper/builds) 直接安装。

## 使用示例

| 输入 | 解析结果 |
|------|----------|
| `午饭三人AA 120元` | 金额120，3人均摊，餐饮 |
| `打车45 我和小王一人一半` | 金额45，2人均摊，交通 |
| `超市买菜200我自己付` | 金额200，单人，购物 |
| `40美元咖啡` | 金额40USD → ¥290，餐饮 |

## 技术栈

- React Native / Expo SDK 56
- expo-sqlite 本地数据库
- 纯离线，零服务器依赖

## 开发

```bash
npm install
npx expo start
```
